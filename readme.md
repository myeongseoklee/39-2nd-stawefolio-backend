# 팀명: Naweke

- 프로젝트 사이트 : **stawefolio**
- 머물고싶은 공간을 큐레이팅하는 스테이폴리오 웹사이트 모델링한 프로젝트
- 고객들의 감성 니즈를 충족하는 숙소 제안 및 제공

## 프로젝트 선정 이유

- 옵션별 숙소 필터링으로 다양한 데이터베이스 활용 경험
- SNS 로그인 및 네이버지도 API 활용 경험

## 구현사항

### [DB modeling] - All

### [Booking]

- Booking API **(POST)** - 은영

### [BookingList]

- Booking List API **(GET)** - 은영
